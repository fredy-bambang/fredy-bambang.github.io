Zero-Downtime MySQL Migration

Duration: Oct 2022 – Dec 2023 | FinAccel (Kredivo)

Solution: Architected near-zero-downtime migration strategy coordinating with Infra, DBA, and Security teams. Led database cloning from Kredivo to Kredifazz Lender Database ensuring OJK financial compliance. Designed real-time audit trail system consuming Kafka streams for configuration change traceability.

Impact: Seamless migration with no service interruption. Improved database performance. Full regulatory compliance maintained.

Event-Driven Insurance System

Duration: 2023 | FinAccel

Challenge: Transform synchronous insurance operations into scalable async flows.

Solution: Pioneered event-driven architecture using Kafka for insurance creation, deactivation, and cancellation. Implemented state-based processing with retry mechanisms and dead-letter queues. Built comprehensive Kafka consumers and scheduled jobs for reliability.

Impact: Reduced blocking operations. Improved system resilience. Enabled better monitoring and debugging of insurance workflows.

Stack: Go, Kafka, MySQL
